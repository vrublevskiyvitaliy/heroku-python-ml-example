We created efficient and simple model for detecting paraphrase sentences in the English language. The dependency tree was chosen as the main structure to represent the relationships between words in a sentence. To represent the word semantics, we are using pre-trained general-purpose word embeddings. Based on these two key components, we designed a few features that can help to identify paraphrases.

Features used:

Hungarian nodes match.

Graph edit distance based on Hungarian nodes match.

Dependency Tree Path Feature.

Dependency Tree Subgraph Feature.

Dependency Tree Subgraph Feature with Idf.

Syntactic N-Grams Overlap.
